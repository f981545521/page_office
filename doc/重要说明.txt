运行demo项目的步骤：
-------------------------------------------------------------------------------------------------
1. 添加PageOffice的jar包（pageoffice4.5.0.2.jar）到本地Maven仓库；
2. 在Eclipse中导入Maven项目：maven-springmvc；
3. 更新项目：Maven - Update Project...
4. 运行项目：Run As - Run on Server
5. 启动浏览器访问：http://localhost:8080/maven-springmvc/demo/index ，即可在线打开、编辑、保存word文件

提示注册的话，请使用PageOffice V4.0 专业版试用序列号：CA1XB-MF7Y-12ST-PSBP2

添加PageOffice的jar包（pageoffice4.5.0.2.jar）到本地Maven仓库：
--------------------------------------------------------------
1 拷贝pageoffice4.5.0.2.jar到D盘根目录；
2 启动CMD命令窗口，输入：D: 回车，切换到D盘根目录下；
3 输入：mvn install:install-file -DgroupId=com.zhuozhengsoft -DartifactId=pageoffice -Dversion=4.5.0.2 -Dpackaging=jar -Dfile=pageoffice4.5.0.2.jar
